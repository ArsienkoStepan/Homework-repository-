Список с парными тешами

  • тег <h1></h1> - заголовок первого уровня
  • тег <p></p> - добавляет параграф
  • тег <div></div> - добавляет блок
  • тег <a></a> - добавляет ссылку
  • тег <strong></storng> - выделяет текст жирным

Список с одиночными тегами

  1. тег <br> - делает перенос строки
  2. тег <hr> - добавляет разделительную линию на всю ширину родителя
  3. тег <input> - добавляет поле (вид завист от того какой тип мы укажем)
  4. тег <img> - добавляет изображение на страницу
  5. тег <link>- загружает внешние файлы, например: CSS

Что такое верстка

Верстка по праву считается одним из самых важных этапов создания сайта. От
качества работы и подготовки, знаний специалиста зависит вся работа, качество
сайта, его вид и функционирования. Даже незначительная ошибка, допущенная в процессе
верстки, может привести к катастрофическим последствиям.
Неправильный код может дать серьезную нагрузку на сайт, неправильно отобразить
компоненты, в результате чего:

  1. посещаемость сайта упадет
  2. прибыль клиента будет стремиться к нулю.
